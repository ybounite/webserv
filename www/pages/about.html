About WebServ
KHEDAM

42 Network Project

HTTP/1.1 Compliant

Full implementation of HTTP/1.1 protocol with support for GET, POST, and DELETE methods. Handles persistent connections, chunked transfer encoding, and proper status code management.

High Performance

Non-blocking I/O architecture using epoll() for concurrent connection handling. Efficient request parsing, optimized memory management, and zero external dependencies.

Enterprise Features

Virtual hosting, CGI script execution, file upload handling, custom error pages, and configuration file parsing. Built for real-world deployment scenarios.

Technical Architecture

Core Technologies

  • ▹ C++98 standard compliance for maximum portability
  • ▹ POSIX socket programming with TCP/IP protocol suite
  • ▹ Multiplexed I/O with select() system call
  • ▹ Custom HTTP parser with RFC 2616 compliance

Advanced Features

  • ▹ Multi-server configuration with virtual hosts
  • ▹ CGI execution for dynamic content generation
  • ▹ Configurable client body size limits and timeouts
  • ▹ Comprehensive error handling and logging system

The Webserv KHEDAM Story

KHEDAM (خدام) — an Arabic word meaning "is working" — represents more than just a project completion. It embodies the journey of transforming theoretical knowledge into a production-ready system through rigorous engineering and collaborative problem-solving.

This server was built through intensive study of network protocols, system-level programming, and software architecture patterns. Every component was carefully designed and implemented from first principles, requiring deep understanding of how web servers operate at the lowest levels.

The development process involved navigating complex technical challenges: implementing non-blocking I/O, parsing HTTP headers correctly, managing socket states, handling edge cases in CGI execution, and ensuring memory safety throughout. Each obstacle was an opportunity to deepen our understanding of systems programming.

The result is a testament to perseverance, technical excellence, and the satisfaction of building something complex and functional from the ground up.
